# Argyle-Landslide
2024 Argyle landslide caused by toe erosion of a slope underlyed by Presumpscot Formation. Repo includes code for processing ADCP transect data into bathymetry &amp; velocities, Cone Penetration Tests for subsurface soil investigation, LiDAR for slide volume &amp; surface mapping, and Arduino code for a lab-built tilt sensor.
